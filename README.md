Hey! I'm Pavan. I started this repository when I decided to seriously sit down and learn C++ properly — not just copy-paste from tutorials, but actually *understand* what's happening under the hood. I've been doing embedded systems stuff with Arduino and STM32 for a while, but I realised my core C++ knowledge had gaps that would eventually catch up with me. This repo is me fixing that, one concept at a time.

I try to push updates every few days. Some weeks more, some weeks less — depends on college and project deadlines. But the goal is to get to a point where I can confidently say: "Yes, I know C++. Not just the syntax — the *why* behind it."

## 💻 Compilation & Setup

### Check g++ Installation
```bash
g++ --version
# Should show: g++ (Ubuntu...) 11.x or similar
```

### Install on Linux
```bash
sudo apt update
sudo apt install g++ build-essential
```

### Compile Any File
```bash
# Basic
g++ HelloWorld.cpp -o hello && ./hello

# With C++17 (recommended for modern features)
g++ -std=c++17 filename.cpp -o output && ./output

# With warnings (I always compile with this)
g++ -std=c++17 -Wall -Wextra -o output filename.cpp
```

### Clone & Run
```bash
git clone https://github.com/PavanShettyHS/CPP-Programming-Mastery.git
cd CPP-Programming-Mastery/01_Basics
g++ -std=c++17 HelloWorld.cpp -o hello && ./hello
```
